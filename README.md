
![](Polly-Logo@2x.png)

我们来介绍一款强大的库Polly，Polly是一种.NET弹性和瞬态故障处理库，允许我们以非常顺畅和线程安全的方式来执诸如行重试，断路，超时，故障恢复等策略。

# 项目示例
运行示例
```sh
git clone https://github.com/scfido/pollyDemo.git
cd pollyDemo
dotnet run
```
